# Green Harvest

A React + Vite farmer app with:
- Welcome screen
- Login and registration with 4 digit PIN
- English and Tamil language choice
- Farmer harvest form
- QR code generation
- Firebase-ready configuration

## Run locally

```bash
npm install
npm run dev
```

Local URL:

```bash
http://127.0.0.1:4173
```

## Firebase

Create a `.env` file from `.env.example` and add your Firebase web app config.
